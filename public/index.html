FCC API: Image Search Abstraction Layer

By Agustina (abrden) Barbetta | GitHub Repository | Free Code Camp API Projects Challenge

User Stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example usage:

To search for 'millenium falcon' images try https://abrden-bing-search.herokuapp.com/imgsearch/millenium falcon You can get 5 results by adding '?offset=5' like https://abrden-bing-search.herokuapp.com/imgsearch/millenium falcon?offset=5
To see the last 10 searches go to https://abrden-bing-search.herokuapp.com/latest

Example output:

Output for the image results [ { url: 'http://...', snippet: '...', thumbnail: 'http://...', context: 'http://...' }, ... ] Output for the most recent searches [ { term: '...', when: '...' }, ... ]
